 Сценарий работы
 1. Создать файл “new.json”
> new.json
 2. Добавить файл под гит
git add new.json
 3. Закоммитить файл
git commit -m "Create new.json"
 4. Отправить файл на внешний GitHub репозиторий
git push
 5. Создать файлы new2.json, new2.xml, new2.txt
> new2.json  new2.xml  new2.txt
 6. Добавить файлы под гит
git add new2.json  new2.xml  new2.txt
 7. Закоммитить файлы
git commit -m "Create new2.json  new2.xml  new2.txt"
 8. Отправить файлы на внешний GitHub репозиторий
git push
 9. Отредактировать содержание файла “new2.txt”
vim new2.txt
I- отредактировать файл
ESC и wq
 10. Отредактировать содержание файла “new2.json”
vim new2.json
I- отредактировать файл
ESC и wq
 11. Добавить и закоммитить “new2.txt” одной строчкой 
git add new_2.txt ; git commit -am "Changed new_2.txt"
 12. Добавить и закоммитить “new2.json” одной строчкой
git add new_2.json ; git commit -am "Changed new_2.json"
 13. Отправить изменения на внешний репозиторий
git push
 14. Отредактировать содержание файла “new2.xml”
vim new2.xml
I- отредактировать файл
ESC и wq
 15. Добавить и закоммитить “new2.xml” одной строчкой
git add new_2.xml ; git commit -am "Changed new_2.xml" 
 16. Отправить изменения на внешний репозиторий
git push
 17. Отправить предыдущее домашнее задание по GitBush Terminal на внешний репозиторий

нажать add file название файла редактировать файл и нажать Commit changes
 19. Синхронизировать внешний и локальный репозитории (слить изменения с внешнего репозитория) 
git pull
 20. Отредактировать в веб интерфейсе содержание файла “new2.json” и сохранить.
нажать на файл потом edit this file(карандашик) внести изменения и нажать Commit changes
